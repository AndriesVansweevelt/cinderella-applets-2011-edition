Man kann auch drei Spiegelebenen so anordnen, dass dabei jede auf der anderen in einem Winkel steht, der ein Teiler von 180° ist. Stellt man drei Spiegel senkrecht auf eine Tischplatte so ergeben sich dafür genau drei verschiedene Möglichkeiten:

  • (60°, 60°, 60°)
  • (90°, 45°, 45°)
  • (90°, 60°, 30°)

Der Grund dafür ist, dass die Winkelsumme im Dreieck immer 180° betragen muss. Bei einem solchen Kaleidoskop ergeben sich Muster, die sich ins Unendliche bruchkantenfrei fortsetzen. In der Simulation nebenan kann man eine solche Struktur beobachten. Am großen Schiebeschalter kann man einstellen, wie oft gespiegelt werden soll. An den Punkten läst sich die Position von Bild und Kaleidoskop verändern.
