Installationsanweisung:


1. Hintergrundinformation:

Die komplette Installation läuft innerhalb eines Web Browsers
Alle notwendigen Dateien sind in diesem Ordner enthalten


2. Installation

Einfach diesen Ordner an eine geeignete Stelle der Festplatte kopieren
(z.B. Desktop oder Home-Verzeichnis)


3. Starten

Öffnen der Datei "index_de.html" mit einem Webbrowser


4. Besonderheiten

a) Das Layout ist optimiert für eine Darstellung im 1920x1080
Vollbildmodus. Der Browser und der Bildschirm sollten entsprechend konfiguriert sein.

b) Idealerweise betreibt man den Browser in einem Kiosk Modus (Linuxeperten fragen, 
wie man das einrichtet). Durch ein geeignetes Script kann dieser direkt 
beim Starten des Rechners aktiviert und gestartet werden

c) Der Screensaver liegt als eine Filmdatei im .ogg Format vor. Diese sollten unter
Linux in Firefox  direkt ansehbar sein. Wenn nicht muss geeignetes 
Plugin installiert werden. Unter anderen Betriebssystemen (z.B. MacOSX) 
kann im Bedarfsfall ein geeignetes 
Format nachgeliefert werden.


5. Voraussetzungen

Ein zeitgemäß schneller Rechner und eine aktuelle Java-Installation.
Bildschirm 1920x1080.
